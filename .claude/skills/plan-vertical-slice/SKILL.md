---
name: plan-vertical-slice
description: Plan a safe end-to-end product slice from user outcome through architecture, tests, telemetry, privacy, and rollout.
---
1. State the user outcome and excluded scope.
2. Read relevant product, architecture, data, clinical, privacy, and UX documents.
3. Map UI, domain, API, persistence, integrations, telemetry, and migration impact.
4. Identify safety, privacy, accessibility, and regulatory impact.
5. Define acceptance criteria and test evidence.
6. Produce a small implementation sequence with rollback points.
