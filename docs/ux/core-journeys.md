# Core User Journeys

## 1. What am I eating?
Entry points: recent meal, favourite, search, barcode, voice, label photo, meal photo, recipe, or manual food. The user confirms quantity and uncertainty before the item enters a meal.

## 2. Build a meal
Add items, change quantities, see carbohydrate and nutrition totals, inspect sources, save as a reusable meal, and operate offline.

## 3. Create a recipe
Add ingredients, record raw or cooked yield, define portions, save variants, and reuse quantities.

## 4. Review history
Inspect, duplicate, edit, export, or delete meals. Clearly distinguish original recorded values from later source updates.

## 5. Caregiver workflow
Switch dependent profiles safely, show identity and profile context persistently, and prevent cross-profile entry.

## 6. Optional calculation review
Show confirmed carbohydrate, glucose source and age, profile settings, active insulin state, arithmetic, warnings, and explicit confirmation. Do not collapse this into a single unexplained number.
