# Definition of Done

A feature requires:
- approved requirement and acceptance criteria;
- architecture and ADR impact addressed;
- implementation and automated tests;
- accessibility review;
- privacy and security impact;
- data provenance and migration impact;
- observability with health-data redaction;
- user-facing error and offline behaviour;
- documentation and support impact;
- rollback plan;
- traceability update;
- independent clinical review when the dose boundary is touched.
